Entendendo as Licenças de Software Livre

Criado por João Paulo Polles

60 - 90 minutos

Nessa atividade os estudantes poderão conhecer um pouco mais sobre licenças de software livre, conhecer os tipos e diferenças entre eles e também saber classificar de qual tipo é uma determinada licença de software livre.

Competências de Alfabetização da Web

Objetivos de aprendizado

  • Entender o que é uma licença de software livre.
  • Entender quais são os tipos e saber diferenciá-los.
  • Saber classificar de qual tipo é uma determinada licença de software livre.

  • 1. Preparação

      O software livre só é livre se o mesmo garantir as 4 liberdades essenciais:

      • Liberdade 0: A liberdade de executar o programa como você desejar, para qualquer propósito.
      • Liberdade 1: A liberdade de estudar como o programa funciona, e adaptá-lo às suas necessidades. Para tanto, acesso ao código-fonte é um pré-requisito.
      • Liberdade 2: A liberdade de redistribuir cópias de modo que você possa ajudar ao próximo.
      • Liberdade 3: A liberdade de distribuir cópias de suas versões modificadas a outros. Desta forma, você pode dar a toda comunidade a chance de beneficiar de suas mudanças. Para tanto, acesso ao código-fonte é um pré-requisito.

      Portanto é interessante escolher alguns softwares para fazer uma diferenciação do que seria o software livre de fato e softwares que são gratuitos para uso, porém, com o seu código fechado.

      Além disso, é interessante estudar o conceito de copyleft, de licenças permissivas, reciprocas parciais e reciprocas totais, além das principais licenças, como MIT, BSD, Apache, LGPL, Mozilla, GPL, AGPL, dentre outras.

      Se você entender o conceito dos tipos de licenças, você saberá encaixar cada uma das licenças de software livre em suas devidas categorias apenas sabendo de suas características.

      Imprima os cartazes que estão nesta pasta para poder fazer a classificação dos tipos de licenças.

  • 2. Introdução

    5 minutos

    Dê as boas-vindas aos participantes da atividade e antes de começar peça para cada um se apresentar e após explique que nessa atividade iremos entender um pouco melhor sobre as licenças de software livre. Nós iremos:

    1. Entender o que é uma licença de software livre.
    2. Entender os tipos de classificação das licenças de software livre.
    3. Conhecer algumas licenças de software livre e classificá-las em seu devido tipo.
